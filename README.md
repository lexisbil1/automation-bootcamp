### 📌 Week 4: Social Media Monitoring & Alerts System 🚨

![Automation](https://img.shields.io/badge/Automation-n8n-orange)
![API](https://img.shields.io/badge/API-NewsAPI-green)
![Category](https://img.shields.io/badge/Type-Monitoring%20System-blue)

A real-time monitoring system that tracks online mentions, analyzes sentiment, and triggers alerts.

**Key Features:**
- ⏱ Automated monitoring every 15 minutes  
- 🌐 News/API-based data ingestion  
- 🧠 Rule-based sentiment classification 
- 🚨 Instant alerts for negative mentions (Telegram)  
- ⭐ Positive mention storage (Airtable)  
- 📊 Centralized logging (Google Sheets)  

**Why This Project Matters:**
This system mimics a **Security Operations Center (SOC)** workflow:
- Data ingestion  
- Event analysis  
- Alerting  
- Logging
